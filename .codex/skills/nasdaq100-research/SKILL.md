# 纳斯达克100研究

## 适用范围

使用本技能处理纳斯达克100专题任务：

- 查询或比较 Nasdaq 100、NDX、QQQ、TQQQ、纳指100 ETF、QDII、ETF联接基金。
- 计算或核验纳指100年度涨跌、基金净值增长率、指数收益、长期收益差。
- 比较纳指100相关基金的申购状态、限额、费用、规模、溢价率、跟踪误差和汇率风险。
- 更新 `基金/纳指100/` 下的本地研究笔记。

不要把 TQQQ 或其他杠杆产品当作普通纳指100指数替代品。讨论杠杆产品时必须说明日内/每日目标杠杆、复利路径依赖、波动损耗和回撤风险。

## 本地知识库

开始工作前读取相关本地文件：

- `基金/投资圣经.md`：用户的长期观察和经验规则。
- `基金/配比.md`：用户当前组合目标、定投安排和已投品种。
- `基金/纳指100/`：纳指100专题历史研究、快照和对比表。

读取 `references/nasdaq100.md` 获取常用来源链接和纳指100专题注意事项。

本地文件名中带日期的内容视为当日快照。引用时要说明快照日期；需要“最新”结论时必须重新核验。

## 资料优先级

1. 官方来源：Nasdaq 指数页面、基金公司产品页、公告、招募说明书、基金合同、定期报告、ETF申赎清单、交易所、监管披露。
2. 理杏仁：用于指数历史涨跌、全收益口径、基金和指数交叉核对。
3. 主流行情或销售平台：只作发现线索或辅助核验，关键字段仍回到官方或理杏仁确认。

如果来源冲突，优先采用官方披露，并在输出中说明冲突点、采用口径和数据日期。若无法联网或无法打开关键来源，明确标注“未能核验最新状态”。

## 工作流

1. 确定任务范围：指数、基金、ETF、QDII、联接基金、杠杆 ETF、时间区间、币种和收益口径。
2. 读取本地纳指100笔记，提取用户偏好、既有结论和上次口径。
3. 核验当前数据：申购/暂停/限额、费率、跟踪标的、基金净值增长率、指数收益、场内溢价率和成交额。
4. 统一口径：价格指数 vs 全收益指数、指数本体 vs 基金净值、人民币 vs 美元、自然年 vs YTD。
5. 计算并复核：所有派生值写明公式；百分比和百分点分开表达；估算值必须标“估”。
6. 输出结论：先给最有用发现，再给表格、口径、风险点和来源。
7. 如需写入文件，保持 `基金/纳指100/` 下的 Markdown 风格，避免覆盖无关内容。

## 运行环境

运行 Python、测试、脚本或数据处理命令时必须使用 `jijin` conda 环境；优先使用 `conda run -n jijin ...`，禁止直接使用系统 `python`/`python3`/`pip`。
